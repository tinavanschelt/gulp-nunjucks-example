# Gulp Nunjucks Example

Basic structure for HTML templating with Nunjucks. Including a gulp-sass-workflow, linting and prettier.

## Gulp workflow

## Basic Setup

1.  Clone / Download
2.  Install dependencies with

```
$ npm install
```

3.Run on `http://localhost:8080/` with

```
$ gulp
```

## Gulp tasks

```
$ gulp watch
```

```
$ gulp lint
```

```
$ gulp build
```
